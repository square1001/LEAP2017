What's this restaurant?

This restaurant started in July 19th on 2017, by Hirotaka Yoneda, for the Codemakers++ program in L.E.A.P McMaster University. In this program, we write HTML and CSS source code, so I decided to make a restaurant to make website :)

I like desserts, so this restaurant has some desserts including cakes, donuts, and fruits. Of course we have some drinks! I really hope you enjoy eating and drinking!
